| `start_at` | `TIMESTAMPTZ` | Non | — | — |
| `end_at` | `TIMESTAMPTZ` | Oui | `NULL` | CHECK |
| `capacity` | `INTEGER` | Non | `1` | CHECK |
| `event_type` | `VARCHAR(20)` | Non | `'OTHER'` | CHECK |
| `theme` | `VARCHAR(120)` | Non | `''` | — |
| `image` | `VARCHAR(100)` | Oui | `NULL` | — |

| `visible` | `BOOLEAN` | Non | `FALSE` | — |

| `client_id` | `BIGINT` | Oui | `NULL` | FK |

```sql
PRIMARY KEY (id)

FOREIGN KEY (organizer_id)
REFERENCES accounts_user(id)
ON DELETE CASCADE

FOREIGN KEY (client_id)
REFERENCES accounts_user(id)
ON DELETE SET NULL

CHECK (
    end_at IS NULL
    OR end_at > start_at
)

CHECK (
    event_type IN (
        'SEMINAR',
        'CONFERENCE',
        'PARTY',
        'OTHER'
    )
)

CHECK (
    status IN (
        'DRAFT',
        'ACCEPTED',
        'IN_PROGRESS',
        'DONE',
        'CANCELLED'
    )
)
```

## 7. Table `crm_quote`

| Colonne | Type PostgreSQL | Null | Défaut | Contraintes |
|---|---|---:|---|---|
| `id` | `BIGINT GENERATED BY DEFAULT AS IDENTITY` | Non | — | PK |
| `client_id` | `BIGINT` | Oui | `NULL` | FK |
| `prospect_id` | `BIGINT` | Oui | `NULL` | FK |
| `status` | `VARCHAR(30)` | Non | `'DRAFT'` | CHECK |
| `tva_rate` | `NUMERIC(5,2)` | Non | `0.20` | CHECK |
| `created_at` | `TIMESTAMPTZ` | Non | — | — |

### Contraintes

```sql
PRIMARY KEY (id)

FOREIGN KEY (client_id)
REFERENCES accounts_user(id)
ON DELETE SET NULL

FOREIGN KEY (prospect_id)
REFERENCES crm_prospect(id)
ON DELETE SET NULL

CHECK (
    status IN (
        'DRAFT',
        'SENT',
        'ACCEPTED',
        'REFUSED',
        'CHANGE_REQUESTED'
    )
)

CHECK (tva_rate >= 0)

CHECK (
    (client_id IS NOT NULL AND prospect_id IS NULL)
    OR
    (client_id IS NULL AND prospect_id IS NOT NULL)
)
```
